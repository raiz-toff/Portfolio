### Key Concepts Covered
- **802.1Q Encapsulation**: Tagging frames for transport across trunk links.
- **VTP (VLAN Trunking Protocol)**: Managing VLAN databases across multiple switches.
- **Router-on-a-Stick (ROAS)**: Using sub-interfaces for efficient Inter-VLAN communication.
- **Access vs. Trunk Ports**: Understanding port membership and tagging requirements.
