## Technical Focus Areas

- **System Programming**: Kernel-level display drivers (EVDI) and low-latency data streaming.
- **Web Development**: Flask backends with SQLite, plus the analytics dashboards built on top of them.
- **Performance**: Using NVENC/VAAPI hardware encoders to keep video encoding off the CPU.
- **Automation**: Developing tools that replace manual workflows and improve daily efficiency.
